# ProiectJavaFoodShop

## Info
This application is a simple implementation of a shop, allows the customer to create baskets and see the invoice for the current transaction at the end.

## Business requirements
1. The applicatiom allows to customer to view all the items availible
2. The applicatiom allows to customer to view each item individual and add it to basket
3. The applicatiom allows to customer to place make an transaction, by which we mean an invoice.
4. The applicatiom allows to administrator to add new items
5. The applicatiom allows to administrator to update the items
6. The applicatiom allows to administrator to delete items
